## Usage

Sample data is automatically loaded when:
- The backend API is not available
- Network requests fail
- You're developing offline

The data is stored in localStorage for persistence across sessions.

## Features

- **Realistic Content**: All sample data uses professional, real-world scenarios
- **Complete Data**: Includes all fields (tasks, team members, dates, statuses)
- **Visual Variety**: Different colors, statuses, and team compositions
- **LocalStorage Sync**: Automatically saves to browser storage
- **Easy Testing**: Great for UI development and testing features

## Customization

You can modify `sampleData.js` to:
- Add more sample projects
- Update team member information
- Change project statuses and timelines
- Add custom resume applications
